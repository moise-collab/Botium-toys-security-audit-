## Botium Toys Security Audit
##  Project Overview
This project presents a security audit conducted for Botium Toys, a growing online business handling customer data and payment processing.

##  Key Focus Areas
- Risk identification
- Security controls evaluation
- Compliance requirements (GDPR, PCI DSS, NIST)

##  Risks Identified
- Weak access controls
- Lack of encryption
- Limited monitoring
- Compliance gaps

##  Recommendations
- Implement MFA
- Strengthen encryption
- Improve monitoring systems
- Ensure compliance with GDPR and PCI DSS

##  Skills Demonstrated
- Security auditing
- Risk analysis
- Cybersecurity frameworks
- Critical thinking
